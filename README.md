Expanded project from the split solver project.
This project is to split bills among roomates.
Based on a bill, this app creates IOUs (transactions), assigned them to payer/debtors.
This IOU can also be consolidated and be shown in account between two people.

This app may have implementations reading PSEG utility bill, Verizon Fios, and some supermarkets' receipts (if they have APIs for an access. If not, maybe I need to use machine learning ughh)

This project wil use MYSQL for DB, Java Spring boot for Backend, Discord/WebApp for frontend.
This app is meant to be hosted on a cloud service, maybe I will use Google Firebase.

This project may be re-written in C++, Go or Python just for a purpose of learning the languages.
This project will be expanded to "RA project" that will also handles common chorse works as well.
